# Project-CV-



*ECG Image Digitization 🌟*
A computer vision project that converts scanned ECG images into digital signals, preserving physiological structure and amplitude relationships of 12 standard ECG leads.

*Overview*
Electrocardiograms (ECGs) are crucial for cardiac diagnosis, but many records exist only as scanned images. This project aims to digitize ECG images into time-series data suitable for automated analysis.

*Features*
- *Image Enhancement*: Improve ECG image quality by reducing noise and enhancing contrast
- *ECG Row Detection*: Automatically detect ECG rows in images
- *Waveform Extraction*: Extract ECG waveforms and correct baseline wander
- *12-Lead Reconstruction*: Reconstruct all 12 standard ECG leads using physiological relationships
- *Signal Post-Processing*: Normalize, smooth, and format signals for submission

*Methodology*
1. Image enhancement and noise reduction
2. ECG row detection and waveform extraction
3. Signal correction and normalization
4. Reconstruction of 12 ECG leads
5. Formatting for submission
